# ASSET-LICENSE-RISK.md

## 1. 왜 이 문서가 필요한가
이 앱은 남자 아이돌 이미지를 핵심 콘텐츠로 사용한다.  
따라서 기술 구현 가능 여부와 별개로 **이미지 사용 권리** 문제가 존재한다.

## 2. 주요 리스크
### 2.1 저작권
언론사 사진, 팬사이트 사진, 방송 캡처 등은 대체로 원저작자가 따로 있다.

### 2.2 초상권 / 퍼블리시티권 유사 이슈
상업성 여부, 공개 범위, 브랜드성 활용 방식에 따라 문제가 될 수 있다.

### 2.3 플랫폼 신고 위험
이미지 권리자가 문제 제기하면 게시 중단 또는 삭제 요구가 들어올 수 있다.

## 3. 상대적으로 안전한 대안
- 직접 촬영 및 사용 허가를 받은 이미지
- 라이선스 확인 가능한 이미지
- 직접 제작한 일러스트
- 실루엣/캐릭터화된 대체 이미지
- 내부 테스트용 비공개 샘플 이미지

## 4. 권장 운영 방식
### 비공개 테스트 단계
- 샘플 이미지로만 기능 검증

### 공개 서비스 단계
- 권리 확인 가능한 자산만 사용
- 출처 기록 유지
- 요청 시 삭제 가능한 운영 프로세스 마련

## 5. 결론
이 문서는 법률 자문이 아니다.  
다만 현재 기획 기준으로는 **기술보다 자산 사용권 정리가 더 큰 리스크**다.
